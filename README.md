# Flipkart Automation Assignment

## Description

This project automates Flipkart website using Selenium WebDriver.

## Automation Flow

* Open Flipkart website
* Close login popup
* Search Bluetooth Speakers
* Apply filters (boAt, 4★ & above)
* Sort price low to high
* Open first product
* Check available offers
* Add to cart
* Navigate to cart page
* Take screenshot

## Tools Used

* Java
* Selenium WebDriver
* ChromeDriver
* Eclipse IDE

## Prerequisites

Before running this project, make sure:

* Java is installed
* Chrome browser installed
* ChromeDriver configured
* Selenium libraries added

## How to Run

1. Open project in Eclipse
2. Navigate to Assignment.java
3. Run the Java file
4. Automation will start
5. Screenshot will be saved as cart_result.png

## Expected Output

* Product added to cart
* Screenshot captured
* Offers count printed in console
